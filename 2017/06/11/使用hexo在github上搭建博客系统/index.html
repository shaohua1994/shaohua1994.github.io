使用hexo在github上搭建博客系统

2017-06-11

date: 2017-06-11 13:07:13

如何使用hexo在github上搭建自己的博客系统呢?

大概可以分为以下几个步骤

  1. 搭建环境准备（包括node.js和git环境，gitHub账户的配置）
  2. 安装Hexo
  3. 配置Hexo
  4. 怎样将Hexo与github page 联系起来
  5. 怎样发布文章
  6. 主题 推荐

1) 搭建环境准备

大概可以分为以下三步

  • Node.js 的安装和准备
  • Git的安装和准备
  • gitHub账户的配置

配置Node.js环境

下载node.js安装文件：

Windows Installer 32-bit
Windows Installer 64-bit

一路next,然后window R 打开cmd命令窗口输入node -v 检查是否安装好了

配置Git环境

下载git安装文件：

GIt官网下载地址：

Git-2.6.3-64-bit.exe

和Node.js一样，大部分设置都只需要保持默认,为了方便在我的电脑环境变量中配置git Path路径,
然后打开命令行输入git -v,如果如下图所示则表示成功

接下来是github账户的注册

打开github，在下图的框中，分别输入自己的用户名，邮箱，密码。

安装Hexo

在自己认为合适的地方创建一个文件夹，这里我以E：/hexo 为例子讲解，首先在E盘目录下创建Hexo文件夹，并在命令行的窗口进入到该目录

在命令行中输入：

npm install hexo-cli -g

然后输入

npm install hexo --save

然后你会看到命令行窗口刷了一大堆白字，下面我们来看一看Hexo是不是已经安装好了。 在命令行中输入：

hexo -v

看到这个说明已经安装好了

hexo的相关配置

初始化Hexo
接着上面的操作，输入：

hexo init

然后输入：

npm install

之后npm将会自动安装你需要的组件

同样是在命令行中，输入：

hexo g

然后输入：

hexo s

然后会提示：

INFO Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.

在浏览器中打开http://localhost:4000/你将会看到：

Hexo在本地的配置已经全都结束了。

hexo如何与github链接

大概分为以下几步

  • 配置git个人信息
  • 配置Deployment

1、设置Git的user name和email：(如果是第一次的话)

git config --global user.name "user name"
git config --global user.email "your email"

2、生成密钥

ssh-keygen -t rsa -C "your email"

配置Deployment

同样在_config.yml文件中，找到Deployment，然后按照如下修改：

deploy:
  type: git
  repo: git@github.com:yourname/yourname.github.io.git
  branch: master

比如我的仓库的地址是git@github.com:gdutxiaoxu/gdutxiaoxu.github.io.git，所以配置如下

deploy:
  type: git
  repo: git@github.com:gdutxiaoxu/gdutxiaoxu.github.io.git
  branch: master

写博客、发布文章

新建一篇博客，执行下面的命令：

hexo new post "article title"

这时候在我的 电脑的目录下 F:\hexo\source\ _posts 将会看到 article title.md 文件

用MarDown编辑器打开就可以编辑文章了。文章编辑好之后，运行生成、部署命令：

hexo g   // 生成
hexo d   // 部署

当然你也可以执行下面的命令，相当于上面两条命令的效果

hexo d -g #在部署前先生成

1）注意需要提前安装一个扩展：
npm install hexo-deployer-git –save

如果没有执行者行命令，将会提醒

deloyer not found:git

部署成功后访问 你的地址，https://yourName.github.io（这里输入你的地址： https://user name.github.io ),将可以看到生成的文章。
